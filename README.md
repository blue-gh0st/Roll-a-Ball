# Roll-a-Ball
A 3d Unity game.


# Build
1. Go to the master branch.
2. Download the directory for your OS
3. Open th directory
4. # Linux:
5. Double-click on the `Roll-a-ball.x86_64`
6. # MacBook:
7. Go to Smash-a-Ball.app
8. Go to Contents
9. Go to MacOS
10. Double-click the Roll a Ball file
