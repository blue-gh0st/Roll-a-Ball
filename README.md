# Roll-a-Ball
A 3d Unity game.


# Build
1. Download the directory for your OS
2. Open th directory
3. # Linux:
4. Double-click on the `Roll-a-ball.x86_64`
5. # MacBook:
6. Go to Smash-a-Ball.app
7. Go to Contents
8. Go to MacOS
9. Double-click the Roll a Ball file
